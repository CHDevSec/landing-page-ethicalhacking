Não é obrigatório, mas é recomendado ter noções básicas de lógica de programação. Durante a mentoria, você aprenderá Python e Bash conforme necessário para os projetos.

Como funcionam as sessões de mentoria?

São sessões 1:1 de 1h30 por semana, via Google Meet. Cada sessão é gravada para revisão posterior. Entre as sessões, você tem suporte via WhatsApp para dúvidas pontuais.

Você desenvolverá um sistema de automação de pentest e um dashboard de vulnerabilidades. Ambos projetos são baseados em cenários reais e ficam no seu portfólio.
